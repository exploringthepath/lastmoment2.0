🔍 STEP 1: Understand the Problem

 Association rules work BEST with nominal (categorical) data

BUT the dataset has:

study_hours ❌ numeric
attendance ❌ numeric

 So we convert them into categories (VERY IMPORTANT)

 STEP 2: Convert Numeric → Nominal
Steps:
Click Filter
Choose:
unsupervised → attribute → Discretize

👉 (Discretize = convert numbers into ranges)

Click on filter name → set:
- bins = 2

👉 (means LOW / HIGH type grouping)

Click Apply

🧩 PART 2: ASSOCIATION RULE MINING



🔄 STEP 4: Go to Associate Tab
Click Associate
⚙️ STEP 5: Select Algorithm

By default you’ll see:

👉 Apriori (already selected ✅)


▶️ STEP 6: Run Algorithm
Click Start
Answer:

Discretization is the process of converting numeric data into categorical data by creating intervals.
