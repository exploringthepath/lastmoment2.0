Q1: What is ARFF file?
ARFF stands for Attribute Relation File Format and is used in WEKA to store datasets.

Q2: What is relation?
Relation represents the name of the dataset.

Q3: What is attribute?
Attribute represents a column or feature in the dataset.

Q4: What is nominal data?
Nominal data represents categorical values like pass or fail.


In this experiment, we loaded a dataset in ARFF format into WEKA Explorer. We explored different attributes, identified numeric and nominal types, visualized their distribution, and applied preprocessing using filters.
